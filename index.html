⚠️ Cross‑Origin Isolation required

ffmpeg.wasm needs SharedArrayBuffer, which is only available when the page is served with special HTTP headers.

If you’re opening this file directly (file://), please start a local web server with the correct headers. Copy one of the commands below and run it in the same folder:

# Python 3
python -c "import http.server; import socketserver; handler = http.server.SimpleHTTPRequestHandler; handler.extensions_map['.wasm'] = 'application/wasm'; httpd = socketserver.TCPServer(('', 8080), handler); print('Serving on http://localhost:8080'); httpd.serve_forever()"
# Node.js (requires npm install -g http-server)
npx http-server -p 8080 --cors -o

After running the server, open http://localhost:8080 and reload this page.
